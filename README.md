# Simon Says Game

A fun and interactive web-based game built with HTML, CSS, and JavaScript. In this game, players are tasked with repeating a sequence of flashing buttons generated by the game, with each level increasing in difficulty. The game continues until the player makes a mistake, and the game over screen displays the player's score.

## Features:
- Interactive button sequence.
- Increasing difficulty with each level.
- User feedback with flashing buttons.
- Game over screen with score display.

## Technologies Used:
- HTML5
- CSS3
- JavaScript

## How to Play:
1. Press any key to start the game.
2. Watch the sequence of flashing buttons.
3. Click the corresponding buttons in the same order.
4. The game ends when you make a mistake, and your score is displayed.
